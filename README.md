## Prerequisites
Vă recomandăm să folosiți **CLion** de la **JetBrains**, aveți subscripție gratuită prin GitHub Student Pack.

- Nu uitați să separați cât mai bine componentele din proiecte, având o structură clean:
    - \> include (header only)
        - \> templates
        - \> types (enums)
        - library.h
    - \> src (source code)
        - library.cpp (implementare)
    - \> data (date CSV sau JSON)
    - CMakeLists.txt
    - main.cpp
